# smartform-backend
Requires Java 17 
Springboot Version 3.0.2
https://docs.spring.io/spring-boot/docs/current/reference/html/getting-started.html

## Getting Started
Install Maven 
https://www.baeldung.com/install-maven-on-windows-linux-mac

Run mvn clean install

To run local server 
mvn spring-boot:run
